Simple Selectors

Type selectors

p{ select tag name }

Class selectors

.class{ select class }

id selector

#id{ select id }

Universal Selectors

* {It selects all tags in our page}

#box *{color:red} ---> This will color all tags of box id

Combinator Selectors

descendent selector

.box p{ this will select p tag of box class. }

p span{ this will select span tag of p element. }

div p span{ this will select span tag of p element which is also in div element. }

direct child selectors

.box h1{ this will select the h1 tag of class box which is direct child. }

adjescent sibling selector

div + p{ this will select immidiate sibling after div.e.g it will select p tag just after div tag. }

General sibling selector

div ~ p{ this will select all sibling p after div. }

Attribute Selectors

below are 7 attribute selector types

  • A[attr]------Element with specific attribute

  • A[attr = val] ------select attribute only whose value is exactly match e.g input[type = 'text'], it will select on input whose type is text.

  • A[attr ^= val] ------It will select only that attribute whose value starts with a particular word.e.g.img[title ='image']

  • A[attr |= val]------ Attribute begins with value OR is first in dashed separated list. Must starting with a value and the value has a hyphen(-). e.g. img[title = 'image-of']

  • A[attr $= val]------ Attribute ends with the value.e.g. if image has value and the value ends with cat then it will be selected.img[title = 'my cat']

  • A[attr *= val]------ value occours anywhere in attribute e.g. img[title = 'of'] this will select all that images whose title has 'of'.

  • A[attr ~= val] ------ value matches attribute in space separated list.img[title = 'of'] it will select that images whose value has of and before and after space is availble.
