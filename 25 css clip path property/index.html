clip-path property

website to make clip paths www.bennettfeely.com/clipy
    clip path property has four values
  1. circle
  2. ellipse
  3. inset
  4. polygon

circle

clip-path : circle (radius at distance-from-x-axis distance-from-y-axis)
clip-path : circle (30% at 50% 50%)

elipse

clip-path : ellipse (horizontal-radius     verticle-radius     at     distance-from-x-axis     distance-from-y-axis)
clip-path : elipse (30% 40% at 50% 50%)

inset

clip-path : inset (top     right     bottom     left)
clip-path : inset (50% 20% 30% 20%)

polygon

we can make as many points as we like in polygon. we can draw any shape we like. i.e. stars.

clip-path : polygon (x-axis,y-axis     x-axis,y-axis     x-axis,y-axis ......)
clip-path : inset (50% 0%, 0% 100%, 100% 100%)
